Beginning

HTML was developed by British scientist Tim Berners-Lee in 1986-1991. The language was created as a means for the exchange of scientific and technical documentation, suitable for use by people without special knowledge in layout and programming.

HTML 1.0

The first version of HTML 1.0 was released in 1991. It included basic tags for creating headings, paragraphs, lists, and links.

HTML 2.0

In 1995, HTML 2.0 was released, which added new elements and attributes, improving the text formatting and image processing capabilities.

HTML 3.2

HTML 3.2, released in 1997, added support for tables, applets, and scripts. This made it possible to create more complex and interactive web pages.

HTML 4.1

In 1999, HTML 4.1 was released, which fixed bugs and improved compatibility with various browsers.

In our time - HTML5

The modern HTML5 standard, released in 2014, has become the new standard for web development. It has added many new elements and APIs, as well as improved support for multimedia and graphical elements.
